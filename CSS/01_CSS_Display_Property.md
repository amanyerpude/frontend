- Block : Takes full width
- Inline : Stays in line
- Flex : Flexible 
- Grid : Two dimensional layout
- None : Disappears

- Block level elements : these take up the full width and start on a new line.
	- div
	- p
	- H1 to H6
	- Header
	- Footer

- Inline Elements : These sit next to other elements on the same line.
	- span
	- a
	- button
	- input
	- img
